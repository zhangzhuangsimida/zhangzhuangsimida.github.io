FLEX弹性盒模型

发表于 2021-03-04 更新于 2021-06-25 分类于 前端 ， css

了解弹性

Flex 是 Flexible Box 的缩写，意为”弹性布局”，可以轻松的控制元素排列、对齐和顺序的控制。

现在的终端类型非常多，使用弹性盒模型可以让元素在不同尺寸终端控制尺寸。

绝大多数设备尤其是移动端都很好的支持FLEX，所以可以放心使用。

声明定义

容器盒子里面包含着容器元素，使用 display:flex 或 display:inline-flex 声明为弹性盒子。

声明块级弹性盒子

flex-direction

用于控制盒子元素排列的方向。

值 描述
row 从左到右水平排列元素（默认值）
row-reverse 从右向左排列元素
column 从上到下垂直排列元素
column-reverse 从下到上垂直排列元素

row-reverse

column-reverse

flex-wrap

flex-wrap 属性规定flex容器是单行或者多行，同时横轴的方向决定了新行堆叠的方向。

选项 说明
nowrap 元素不拆行或不拆列（默认值）
wrap 容器元素在必要的时候拆行或拆列。
wrap-reverse 容器元素在必要的时候拆行或拆列，但是以相反的顺序
